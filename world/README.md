## Instructions
In a file called `hello.c`, in a folder called `world`, implement a program in C that prints `hello, world\n`, and that’s it!

## How to Begin

Open the terminal window at the bottom of your screen. You should find that your terminal prompt looks something like this:

```bash
student@codespace:/workspaces/codingrabbit-cs50-week1$
```

Next, execute the following command to change into the `world` directory where you will write your program:

```bash
cd world
```

Your terminal prompt should now update to show you are inside the folder:

```bash
student@codespace:/workspaces/codingrabbit-cs50-week1/world$
```

You can now execute the `code` command to create a new file and open it in the editor:

```bash
code hello.c
```

This will create a blank file called `hello.c` where you can write your C code!

## How to Test

Recall that you can compile your `hello.c` program by running:

```bash
make hello
```

If you don’t see an error message, it compiled successfully! You can confirm this by running:

```bash
ls
```

This command lists the files in your folder. You should see not only `hello.c` (your source code) but also `hello` (the compiled machine code).

If you **do** see an error message, don't panic! Try to read the error and fix your code, then try to compile it again. 

If you don't understand the error message, **click the Coding Rabbit icon** on the left sidebar and ask your AI Teaching Assistant for help! (Make sure you are signed in first).
